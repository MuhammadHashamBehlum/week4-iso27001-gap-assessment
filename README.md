## Overview

This repository contains my Week 4 internship assignment that was completed during the Sohail Smart Solutions for the Summer Training Program 2026.

The objective of this assessment was to be able to evaluate the information security posture of a fictional mid-sized organization against the selected ISO/IEC 27001:2022 Annex A controls. The assessment is able to identify the implementation gaps, as well as evaluate the control maturity, and even recommend the remediation activities that are able to strengthen the organization's Information Security Management System (ISMS).

## Assessment Scope

The assessment reviews **15 Annex A controls** across the following four security themes:

- Organizational Controls
- People Controls
- Physical Controls
- Technological Controls

## Repository Contents

- **gap-assessment.md** – Executive summary of the ISO/IEC 27001 gap assessment.
- **assessment-table.csv** – Assessment table containing the evaluated Annex A controls.

## Key Findings

- The information security policies tend to require regular reviews, as well as formal approval.
- The cloud security governance should be strengthened through the documented processes.
- The privileged access rights do require periodic review.
- The Multi-factor authentication should be expanded to all of the users.
- Continous observation should include cloud environments, as well as automated alerting.

## Key Recommendations

1. Make sure to apply and implement the quarterly privileged access reviews thoroughly.
2. Ensure to expand the multi-factor authentication across the business systems.
3. Make sure to set up and establish a formal cloud security governance.
4. Improve the SIEM monitoring, as well as automated alerting.
5. Conduct and establish the annual policy reviews, as well as the periodic internal audits.
